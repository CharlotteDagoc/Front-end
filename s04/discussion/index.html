HTML Images

An image can be shown using <img> tag along with its src attribute.

<img src="https://t4.ftcdn.net/jpg/00/97/58/97/360_F_97589769_t45CqXyzjz0KXwoBZT9PRaWGHRk5hQqQ.jpg ">

<img src="./images/clark.png">

Valid image extensions are the following:

  1. png
  2. jog
  3. gif
  4. svg
  5. webp
